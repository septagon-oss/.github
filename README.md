# .github

This is the special repository for the `septagon-oss` GitHub organization.

It serves as the org-wide default for:

- **Profile README** — the landing page visitors see at `github.com/septagon-oss`
- **Issue templates** — bug report and feature request templates for every repo
- **Pull request template** — default PR template for every repo
- **Code of Conduct** — Contributor Covenant v2.1
- **Contributing guide** — default contribution expectations
- **Security policy** — vulnerability reporting path
- **Support guide** — community and commercial support channels

Repositories that define their own versions of these files will override these
defaults. See [GitHub's documentation](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file)
for details.
